Welcome

API Endpoints

POST /parse-receipt
Upload a single receipt image for OCR processing
POST /parse-receipt-url
Parse a receipt from an image URL
POST /parse-receipts
Batch process multiple receipt images (up to 50)
POST /parse-receipts-url
Batch process receipts from multiple URLs
POST /parse-drive-folder
Process all images in a Google Drive folder
POST /verify-receipts
Verify receipt images against a transaction history PDF
POST /verify-parsed
Verify pre-parsed receipts against transaction history
GET /health
Health check endpoint

Getting Started

To use this API, send a POST request to one of the endpoints with your receipt image. Visit the interactive documentation to try it out directly in your browser.

The API returns structured JSON with transaction details including reference numbers, amounts, timestamps, and OCR confidence scores. Perfect for automating receipt verification and record-keeping.
